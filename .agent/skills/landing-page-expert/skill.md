# Skill: High-Conversion Landing Page Expert

### **Purpose**

This skill enables the agent to conceptualize, structure, and generate copy for high-conversion, mobile-first landing pages. It synthesizes principles from user psychology, visual design, narrative storytelling, and technical optimization to create a persuasive user experience where conversion is the most logical outcome.

### **Core Capabilities**

When activated, this skill will apply the following principles to any landing page generation request:

1.  **Single Goal Focus:** The agent will always prioritize a single, clear Call-to-Action (CTA) for the entire page to eliminate confusion and friction.

2.  **Mobile-First Architecture:**
    *   Designs are structured for a seamless vertical scroll on a mobile device first.
    *   Ensures a **thumb-friendly UI**, placing primary CTAs and interactive elements within the natural thumb zone.
    *   Mandates large, tappable buttons (min. 44x44 pixels) and legible font sizes (min. 16px).

3.  **Psychological Persuasion Engine:**
    *   **Loss Aversion:** Integrates urgency (countdown timers, limited-time language) and scarcity ("Only X left!") to leverage FOMO.
    *   **Social Proof:** Strategically places testimonials, client logos, and user statistics to build trust and credibility.
    *   **Authority:** Establishes the brand as a credible expert or guide.

4.  **Narrative Structure:**
    *   Applies proven storytelling frameworks (e.g., **Problem-Agitate-Solve**, **Hero's Journey/StoryBrand**) to create an emotional connection with the visitor.
    *   Ensures copy is **"you-focused,"** emphasizing benefits over features.

5.  **Visual & Layout Hierarchy:**
    *   Structures the "Above the Fold" section for immediate impact, including a benefit-driven headline, UVP, and primary CTA.
    *   Uses principles of size, contrast, and whitespace to guide the user's eye.
    *   Applies knowledge of **Z-Pattern and F-Pattern** reading behaviors to inform layout.
    *   Leverages **Color Psychology** to select a palette that evokes the desired emotion (e.g., blue for trust, red/orange for urgency).

6.  **Frictionless Conversion Points:**
    *   Designs simple, single-column forms that only ask for essential information.
    *   Includes trust signals (privacy policies, security badges) near forms.
    *   Crafts compelling, action-oriented CTA button copy.

7.  **Technical Performance Standards:**
    *   All generated page concepts are designed with a target load speed of **under 3 seconds**.
    *   Recommends technical best practices like image compression (WebP), code minification, and browser caching.

### **Inputs**

This skill requires the following inputs from the user to function effectively:
*   `primary_goal`: The single conversion objective.
*   `target_audience_profile`: A description of the ideal customer and their pain points.
*   `product_service_info`: Key benefits and features of the offering.
*   `available_social_proof`: Any existing testimonials, client logos, or case studies.

### **Output**

A structured prompt or a complete draft of a landing page that is technically sound, psychologically persuasive, and optimized for mobile conversion.
