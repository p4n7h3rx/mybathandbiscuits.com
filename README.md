# Bath & Biscuits — Luxury Pet Grooming & Boutique

A premium, high-end web experience for Bath & Biscuits, featuring modern animations, responsive design, and an integrated boutique experience.

## 🚀 Getting Started

### Prerequisites
- **Node.js:** v20.0.0 or higher (Required for React Router v7)
- **npm:** v10.0.0 or higher

### Installation
1. Clone the repository:
   ```bash
   git clone git@github.com:p4n7h3rx/mybathandbiscuits.com.git
   cd mybathandbiscuits.com
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```

### Production Build
Generate optimized artifacts for deployment:
```bash
npm run build
```
The build artifacts will be located in the `dist/` folder.

## 🛠 Tech Stack
- **Framework:** React 18 (Vite)
- **Styling:** Tailwind CSS
- **Animations:** GSAP, Framer Motion
- **3D Features:** Three.js (React Three Fiber)
- **Routing:** React Router v7

## 📂 Project Structure
- `src/`: Source code including components, sections, and pages.
- `public/`: Static assets (images, favicon, etc.).
- `dist/`: Production-ready build artifacts.
- `index.js`: Standard Node.js entry point for production servers (Express).

## 📡 Deployment (Hostinger)
The project is configured for automated deployment via GitHub. 
- **Entry Point:** `index.js`
- **Port Binding:** `0.0.0.0`
- **Node Version:** Ensure your hosting environment is set to **Node.js 20**.
